# Car Manager

## Description

Car Manager is a car and service management program that allows you to add, search, delete, and display information about cars and services.

## Features

* Add cars with detailed information (name, model, year, make, color, number of doors, engine type)
* Search for cars by ID
* Delete cars by ID
* Show all cars
* Add services with detailed information (name, description, and price)
* Search for services by ID
* Delete services by ID
* Show all services

## Requirements

* Python 3.x

## Usage

1. Run the `main.py` file
2. Follow the on-screen instructions to add, search for, delete, and show cars and services

## Project Structure

* `car.py`: `Car` class representing a car
* `car_manager.py`: `CarManager` class managing the car list
* `service.py`: `Service` class representing a service
* `service_manager.py`: `ServiceManager` class managing the service list
* `manager.py`: `Manager` class managing the business logic
* `main.py`: Main file that runs the program
